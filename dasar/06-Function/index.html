## üß† Apa Itu Fungsi?

Fungsi = blok kode yang bisa dipakai berulang-ulang.  
Bisa diibaratkan kayak mesin otomatis buat tugas tertentu.

```javascript
function sapa() {
  console.log("Halo, Marno!");
}

sapa(); // Output: Halo, Marno!

    üß± Struktur Fungsi

    function namaFungsi() {
  // isi perintah di sini
}

    Panggil dengan: namaFungsi();

    üì• Fungsi dengan Parameter

    function sapa(nama) {
  console.log("Halo, " + nama + "!");
}

sapa("Marno"); // Output: Halo, Marno!
sapa("Putri"); // Output: Halo, Putri!

    üì§ Fungsi dengan Return Value

    function tambah(a, b) {
  return a + b;
}

let hasil = tambah(3, 5);
console.log(hasil); // 8

    üß† Function Expression

    Fungsi bisa disimpan dalam variabel juga:

    const kali = function(x, y) {
  return x * y;
};

console.log(kali(4, 5)); // 20

    ‚ö° Arrow Function (Versi Singkat)

    Arrow function = gaya modern, lebih ringkas:

    const bagi = (a, b) => {
  return a / b;
};

console.log(bagi(10, 2)); // 5

    Kalau cuma 1 baris:

    const halo = nama => `Halo, ${nama}!`;
console.log(halo("Marno")); // Halo, Marno!

    üß† Quiz Mini

    Apa hasil dari kode berikut?

    function cek(nama) {
  return "Halo " + nama;
}
console.log(cek("Asep"));
    • A. Halo Asep
    • B. cek Asep
    • C. undefined
    • D. Error

    ‚úÖ Jawaban: A. Halo Asep

    üí™ Tantangan Coding

    1. Buat fungsi luasPersegi(sisi) ‚Üí return luas
    2. Buat fungsi luasSegitiga(alas, tinggi) ‚Üí return luas
    3. Buat fungsi cekGanjilGenap(angka) ‚Üí return 'Ganjil' atau 'Genap'

    Bonus: Gunakan arrow function buat semuanya ‚ö°

    üß† Tips Tambahan

    • Fungsi bikin kode lebih rapi, reusable, dan mudah diuji
    • Jangan takut pecah-pecah kode ke fungsi kecil
    • Kombinasikan fungsi dengan loop atau kondisi buat skill kombo! üåÄ
    ```
