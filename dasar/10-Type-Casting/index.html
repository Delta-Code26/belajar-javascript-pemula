## üß† Apa Itu Type Casting?

**Type casting** adalah proses mengubah satu tipe data ke tipe lain.

Contoh:
```javascript
let angka = "42";
let hasil = Number(angka); // dari string ke number

    1Ô∏è‚É£ Implicit Type Casting (Coercion)

    JavaScript kadang otomatis ubah tipe data saat dibutuhkan.

    Contoh:

    let hasil = "5" * 2;    // hasil: 10 ‚Üí "5" diubah jadi number
let gabung = "5" + 2;   // hasil: "52" ‚Üí angka diubah jadi string

    ‚ö†Ô∏è Hati-hati!
    + bisa berarti penjumlahan atau penggabungan string.

    2Ô∏è‚É£ Explicit Type Casting

    Kamu ubah tipe data secara langsung pakai fungsi:

    ‚û°Ô∏è String ‚Üí Number

    Number("42");        // 42

    ‚û°Ô∏è Number ‚Üí String
    ```
